* <span data-ttu-id="97c27-121">У токенов короткий срок жизни (по умолчанию — один час), что ограничивает распространение атаки.</span><span class="sxs-lookup"><span data-stu-id="97c27-121">Tokens have a short lifetime, one hour by default, which limits the attack window.</span></span> <span data-ttu-id="97c27-122">Токены можно отозвать в любое время.</span><span class="sxs-lookup"><span data-stu-id="97c27-122">Tokens can also be revoked at any time.</span></span>

* <span data-ttu-id="97c27-126">Токены с OAuth и OIDC обеспечивают безопасность приложения вне зависимости от поведения агента пользователя.</span><span class="sxs-lookup"><span data-stu-id="97c27-126">Tokens with OAuth and OIDC don't rely on the user agent behaving correctly to ensure that the app is secure.</span></span>

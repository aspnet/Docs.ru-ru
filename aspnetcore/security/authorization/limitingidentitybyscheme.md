## <a name="selecting-the-scheme-with-policies"></a><span data-ttu-id="b695c-124">Выбор схемы с политиками</span><span class="sxs-lookup"><span data-stu-id="b695c-124">Selecting the scheme with policies</span></span>

<span data-ttu-id="b695c-125">Если вы предпочитаете указывать нужные схемы в [политике](xref:security/authorization/policies), вы можете задать `AuthenticationSchemes` коллекцию при добавлении политики:</span><span class="sxs-lookup"><span data-stu-id="b695c-125">If you prefer to specify the desired schemes in [policy](xref:security/authorization/policies), you can set the `AuthenticationSchemes` collection when adding your policy:</span></span>
